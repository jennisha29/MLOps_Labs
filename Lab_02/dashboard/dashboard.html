This lab demonstrates how Apache Airflow can be used to run a complete machine learning workflow step by step.

What was done in this lab:

  • Built an Airflow DAG to run the entire workflow step by step
  • Loaded the e-commerce customer dataset as the first task
  • Cleaned and scaled the data for clustering
  • Trained a KMeans model and used the elbow method to find the best number of clusters
  • Saved the trained model and clustering results
  • Generated visual outputs like the elbow curve and cluster scatter plot
