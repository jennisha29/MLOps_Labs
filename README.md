# MLOps Labs

A collection of hands-on MLOps labs covering testing, orchestration, containerization, APIs, experiment tracking, and Kubernetes deployment.

| Lab | Topic | Key Technologies |
| --- | ----- | ---------------- |
| Lab_01 | Testing & CI/CD | pytest, unittest, GitHub Actions |
| Lab_02 | Workflow Orchestration | Apache Airflow, Docker Compose, K-Means |
| Lab_03 | Docker & ML Training | Docker, Gradient Boosting, Flask, TensorFlow |
| Lab_04 | REST API + Frontend | FastAPI, Ridge Regression, nginx, Docker Compose |
| Lab_05 | Experiment Tracking | Weights & Biases, XGBoost, Keras CNN |
| Lab_06 | Kubernetes & GKE | GKE, kubectl, K8s Manifests, FastAPI, LoadBalancer, Autoscaling |
